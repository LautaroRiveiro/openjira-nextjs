## Comandos
- **Instalar dependencias:** `npm install`
- **Correr en desarrollo:**
  - Iniciar mongo vía docker:
    `docker-compose up -d`
  - Iniciar frontend
    `npm run dev`
  - MongoDB URL local:
    `mongodb://localhost:27017/entriesdb`
